# lamp5.3.29-xdebug

## Overview

Old version PHP docker set.  
Use docker-compose.  
Including XDebug.

- Debian
- Apache 2.4.10
- Mysql 5.1.73
- PHP 5.3.29

## Usage

```bash
git clone git@github.com:NickelLab/lamp5.3.29-xdebug.git
```
```bash
docker-compose up -d
```
document root is **code** directory.

### Xdebug

The port used by xdebug is 9001.
```ini
xdebug.remote_port=9001
```

### Mysql
mysql root password is '**password**'
